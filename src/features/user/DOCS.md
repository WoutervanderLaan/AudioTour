# src/features/user

User management and profile feature.

## Purpose

Handles user profile and account management including:

- User profile display and editing
- User preferences
- Account settings
- User data management

## Structure

- **components/** - User-specific UI (profile forms, settings, etc.)
- **hooks/** - User hooks (useUser, useProfile, etc.)
- **services/** - User business logic and API calls
- **types/** - User-related TypeScript types

## Import Rules

Can only import from: shared, store
