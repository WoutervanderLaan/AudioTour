# src/shared/components/features/add-photo

Pressable button component for adding new photos to the tour.

## Purpose

Provides an **AddPhoto** component - a pressable button with a dashed border and '+' symbol that triggers photo capture or selection.

## Features

- Three size variants: `sm` (60px), `md` (100px), `lg` (140px)
- Dashed border styling to indicate an action area
- Extends PressableBase for full pressable functionality
- Integrates with the app's theme system
