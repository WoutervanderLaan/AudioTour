# src/shared/components/features

Complex reusable feature components.

## Features

- **[add-photo/](./add-photo)** - Photo selection and upload component
- **[audio-player/](./audio-player)** - Audio playback component with controls
- **[banner/](./banner)** - Banner notification components
- **[progress-indicator/](./progress-indicator)** - Progress indicator component for multi-step flows
- **[thumbnail/](./thumbnail)** - Image thumbnail display component
- **[toast/](./toast)** - Toast notification components

## Purpose

Contains complex, self-contained UI features that are reusable across the app but are more sophisticated than simple UI components. These components:

- May have internal state management
- Provide complete functionality (not just presentation)
- Are still generic (not tied to specific business features)
