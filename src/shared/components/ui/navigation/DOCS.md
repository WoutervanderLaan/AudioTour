# src/shared/components/ui/navigation

Navigation-related UI components.

## Components

- **NavItem.tsx** - Reusable navigation list item component
- **NavItem.stories.tsx** - Storybook stories for NavItem

## Purpose

Provides reusable navigation UI components for settings screens, profile screens, and other navigation-heavy interfaces. The NavItem component is a consistent way to display navigation options with icons, labels, and chevrons.
