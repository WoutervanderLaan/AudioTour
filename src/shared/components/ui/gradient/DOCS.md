# src/shared/components/ui/gradient

Gradient background components for visual effects and testing.

## Files

- **GradientBackground.tsx** - Linear gradient background component with customizable colors and direction
- **GradientBackground.stories.tsx** - Storybook stories for the gradient component

## Purpose

Provides a reusable gradient background component primarily used for testing blur effects and visual styling. The component wraps Expo's LinearGradient with a simplified API and sensible defaults for common use cases.
