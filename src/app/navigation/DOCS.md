# src/app/navigation

Navigation configuration and setup.

## Contents

- **index.tsx** - Main navigation structure with bottom tabs and native stack navigator
- **linking.ts** - Deep linking configuration for the app
- **[screens/](./screens)** - Screen components used in navigation

## Navigation Structure

- **Root Stack**: ObjectDetail, Narrative, Settings (modal), NotFound
- **Home Tabs**: Capture, Museum, Recommendations

Uses React Navigation 7 with static TypeScript configuration for type-safe navigation.
