# src/core

Core infrastructure and utilities used throughout the application.

## Structure

- **api/** - API client, query configuration, types, and MSW mocks for data fetching
- **lib/** - Shared utility functions and helpers (logger, etc.)
- **navigation/** - Navigation configuration, routing, and type definitions

## Purpose

This folder contains the foundational building blocks that other parts of the app depend on. It includes the API client setup with TanStack Query, navigation infrastructure using React Navigation, and common utilities like logging.
